## ✨ Features

- **Swipe** (mobile) + **Arrow keys** (desktop) to move tiles
- **Merge equal tiles** — 2+2=4, 4+4=8... reach **2048** to win!
- **Score + Best Score** — personal best saved across sessions
- **Kids & Adult modes** — different color palettes, same great game
- **Dark mode** — respects system preference automatically
- Zero dependencies · Works offline · Single HTML file
