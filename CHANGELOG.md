# Changelog

## [1.1.0] — 2026-07-04

### Added
- Kids mode: genuine 3×3 grid (instead of 4×4)
- Kids mode: win target reduced to 256 (instead of 2048)
- Kids mode: Undo button — revert the last move
- Kids mode: no loss state — keep playing as long as you like
- Hint bar updates dynamically for each mode

## [1.0.0] — 2026-07-04

### Added
- Initial release of 2048
- Kids mode: pastel tile colors
- Adult mode: classic 2048 colors
- Arrow key + swipe controls
- Score and best score (localStorage)
- Win detection at 2048, loss detection when no moves remain
- How-to-play hint bar
- Dark mode via prefers-color-scheme
- SVG social preview image (1200×630)
- GitHub Pages deployment
